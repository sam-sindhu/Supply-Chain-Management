# Supply-Chain-Management
A data-driven Supply Chain Management project built using Excel, Power BI, Tableau, and SQL to analyze operational performance, track logistics efficiency, and optimize the end-to-end supply chain workflow.

Tools & Technologies
Excel: Data cleaning, preprocessing, pivot analysis
SQL: KPI generation, joins, aggregations, stored queries
Power BI: Interactive dashboards, DAX measures
Tableau: Visual storytelling and exploratory dashboards
